# wrapper
All files from anonymous, No need for an Android emulator to decrypt alac


### 安装
推荐[Windows Subsystem for Linux (WSL)](https://github.com/bostrot/wsl2-distro-manager)使用
usage:  `./wrapper [port] ([username] [password])`

```shell
sudo -i
wget "https://github.com/itouakirai/wrapper/releases/download/linux/wrapper.linux.x86_64.tar.gz"
mkdir wrapper
tar -xzf wrapper.linux.x86_64.tar.gz -C wrapper
cd wrapper
./wrapper
```
